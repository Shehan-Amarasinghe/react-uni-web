
# React-uni-web

This is a fully resposive React based university website.


## Technologies Used

 - React
 - Pure CSS
 - React Router
 - React Icons

## Usage

https://shehan-amarasinghe.github.io/react-uni-web/


## Deployment

To get started, just clone the repository and run npm install && npm run dev

```bash
  npm install
  npm run dev
```
